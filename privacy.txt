Privacy Policy for Cmp Eax, 42

This app does not store any kind of information or share information with its creator.

However, when making requests to the Compiler Explorer API and create links, the Compiler Explorer Privacy Policy applies.
The policy can be read on the website https://godbolt.org/ or directly https://godbolt.org/policies/privacy.html

Be aware that the app will make at least one request to the API when you run it.
And most, if not all functionality is linked to the API.
If you do not agree with the Compiler Explorer Privacy Policy, do not use this app.

All API requests are sent over HTTPS directly to the Compiler Explorer API.
